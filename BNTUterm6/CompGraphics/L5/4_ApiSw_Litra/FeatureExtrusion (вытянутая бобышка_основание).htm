Метод FeatureExtrusion (вытянутая бобышка/основание)

Для того чтобы создать элемент вытянутая бобышка в SW api существует метод FeatureExtrusion2 интерфейса IFeatureManager.

Фактически этот метод соответствует действию кнопки «вытянутая бобышка» в среде SW.

Он принимает целых 23 параметра. Рассмотрим их подробнее.

<FeatureExtrusion2 Sd, Flip, Dir, T1, T2, D1, D2, Dchk1, Dchk2, Ddir1, Ddir2, Dang1, Dang2, OffsetReverse1, OffsetReverse2, TranslateSurface1, TranslateSurface2, Merge, UseFeatScope, UseAutoSelect, T0, StartOffset, FlipStartOffset>

1. Sd. Тип Boolean. True – вытягивание в одном направлении. False – в двух направлениях.

2. Flip. Тип Boolean. True – переставить сторону для выреза. Не применяется для элемента вытягивания.

3. Dir. Тип Boolean. True – поменять направление вытягивания. По умолчанию, выдавливание элемента происходит по направлению нормали.

4. T1. Тип long. Граничное условие для вытягивания элемента по направлению 1. Число соответствует именованной константе из enum-списка swEndConditions_e.

  • 0 – swEndCondBlind – никакие граничные условия не применяются. Выдавливаем элемент на заданное расстояние.
  • 1 – swEndCondThroughAll – через всю существующую геометрию
  • 2 – swEndCondThroughNext – вырез до следующей грани или поверхности. Поверхность должна заслонять весь профиль эскиза и находится в той же детали.
  • 3 – swEndCondUpToVertex – выдавливание или вырез до вершины. Объект вытягивается до плоскости, параллельной плоскости эскиза, и проходящей через указанную вершину.
  • 4 – swEndCondToSurface – вытягивание или вырез до поверхности
  • 5 – swEndCondOffsetFromSurface – вытягивание или вырез на заданное расстояние от поверхности.
  • 6 – swEndCondMidPlane – вытягивание или вырез от средней плоскости. Другими словами симметрично.
  • 7 – swEndCondUpToBody – вытягивание или вырез до тела.

5. T2. Тип long. Граничное условие для вытягивания элемента по направлению 2. Аналогично T1. Если в Sd стоит true (вытягивание в одном направлении), то данный параметр игнорируется.

6. D1. Тип double. Глубина выдавливания по направлению 1 в метрах. Если в T1 (граничное условие) находится что-то отличное от 0, 5 или 6, то данный параметр игнорируется.

7. D2. Тип double. Глубина выдавливания по направлению 2 в метрах. Аналогично D1. Чтобы сработало должно быть в Sd – false(вытягивание по двум направлениям) и в T2 – 0 (никаких граничных условий) , 5 (на расстоянии от поверхности) или 6 (от средней плоскости).

8. Dchk1. Тип Boolean. True – разрешить сделать уклон по направлению 1.

9. Dchk2. Тип Boolean. True – разрешить сделать уклон по направлению 2. Если в Sd – false, то данный параметр игнорируется.

10. Ddir1. Тип Boolean. True – для направления 1 сделать уклон вовнутрь, false – уклон наружу. Если в Dchk1 находится false, то данный параметр игнорируется.

11. Ddir2. Тип Boolean. True – для направления 2 сделать уклон вовнутрь, false – уклон наружу. Аналогично Ddir1. Если в Sd – true, или в Dchk2 – false, то этот параметр игнорируется.

12. Dang1. Тип double. Угол уклона для направления 1. Значение в радианах. Игнорируется если в Dchk1 стоит false(запрет уклона).

13. Dang2. Тип double. Угол уклона для направления 2. Игнорируется если в Dchk2 – false, или в sd – true(вытягивание в одном направлении).

14. OffsetReverse1. Тип Boolean. Сработает, если вы выбрали смещение (на расстоянии от поверхности) первым граничным условием. True – смещение в направлении от эскиза. False – смещение в направлении к эскизу.

15. OffsetReverse2. Тип Boolean. Аналогично offsetReverse1, только для направления2.

16. TranslateSurface1. Тип Boolean. Этот параметр сработает, если в T1(граничное условие) поставить 5 (вытягивание на заданное расстояние от поверхности). True – мы должны пересечь указанную поверхность на расстояние указанное в D1. False – не дойти до поверхности на это расстояние.

17. TranslateSurface2. Тип Boolean. Аналогично TranslateSurface1, только для направления 2. Игнорируется если в Sd – true, или в T2 стоит что-то кроме 5.

18. Merge. Тип Boolean. True – объединить результат в одно тело. False – элемент выдавливания будет отдельным телом.

19. UseFeatScope. Тип Boolean. True – если метод должен воздействовать только на выделенные тела, false – действие распространяется на все тела.

20. UseAutoSelect. Тип Boolean. Если стоит false, то пользователь должен выделить тела, на которые будет действовать метод.

21. T0. Тип long. Стартовые условия. Число из enum-списка swStartConditions_e

  • 0 – swStartSketchPlane –вытягивание от плоскости эскиза
  • 1 – swStartSurface – вытягивание от указанной поверхности, грани или плоскости
  • 2 – swStartVertex – вытягивание от указанной вершины.
  • 3 – swStartOffset – Смещение начала выдавливания на определенное расстояние.

22. StartOffset. Тип double. Если в T0 стоит 3 (смещение), тогда здесь указываем значение этого смещения в метрах.

23. FlipStartOffset. Тип Boolean. Сработает если в T0 указана 3 (swStartOffset). True – поменять направление. По умолчанию смещение происходит по направлению нормали эскиза.

Обращаю внимание, что все элементы, тела, поверхности, которые могут понадобиться при создании данного элемента построения, должны быть выделены и отмечены заранее.

Теперь рассмотрим несколько примеров.

Вводная часть программы. Сначала получаем доступ к соответствующим интерфейсам.

Чтобы воспользоваться методом FeatureExtrusion2, нам надо получить доступ к интерфейсу IFeatureManager. Сделать это можно с помощью свойства FeatureManager интерфейса IModelDoc2.

Задача1

Есть замкнутый эскиз прямоугольника. Эскиз выделен в дереве. Надо его вытянуть на 50 мм.

Visual Basic
1
2
3
4
Dim feat As Feature
Set feat = sel.GetSelectedObject6(1, -1)
fm.FeatureExtrusion2 True, False, False, 0, 2222, 0.05, 222, False, 3, False,
False, 131231, 23131, False, False, False, False, False, False, False, 0, 553532554, False

Сначала получаем выделенный объект (у нас это эскиз) и далее производим операцию вытягивания.

Большинство указанных параметров нам не важны. Первым параметром (Sd) указываем true – выдавливание в одном направлении. Значит все, что касается второго направления можно проигнорировать. Проигнорировать – это значит проставить любые значение, которые соответствуют типу.

Четвертый параметр (T1) – граничное условие для направление1. Ставим 0 (никаких граничных условий).

Шестой параметр (D1) – на какую длину вытягивать по направлению 1. Здесь мы указываем наши 50мм (в метрах это 0,05).

Третий параметр с конца (T0) – Стартовые условия. Ставим 0 – вытягивание от плоскости эскиза.

Задача2

Есть плоскость с эскизом на некотором расстоянии от другого тела. Надо вытянуть элемент до указанной поверхности тела. Перед вызовом макроса последовательно выделяем, при нажатом Ctrl, эскиз в дереве конструирования (FeatureManager), затем грань.

Dim feat1 As Feature
Dim face As Face2
Set feat1 = sel.GetSelectedObject6(1, -1)

fm.FeatureExtrusion2 True, False, False, 4, 2222, 0.05, 222, False, 3, False, False, 131231, 23131, False, False, False, False, False, False, False, 0, 553532554, False

Сначала «ловим» выделенные объекты методом GetSelectedObject6, затем создаем вытянутый объект.

По сравнению с задачей1, в методе extrusion поменялся только 4-й параметр (T1) – граничное условие для вытягивания в направлении 1. Ставим 4 – swEndCondToSurface – вытягивание или вырез до поверхности. Конкретная высота выдавливания значения уже не имеет.

Задача3

Есть плоскость с эскизом на некотором расстоянии от другого тела. Надо вытянуть элемент от противоположной грани на расстояние 30 мм. Перед вызовом макроса последовательно выделяем при нажатом Ctrl эскиз, затем грань.

Решение мало отличается от задачи2, за исключением пары параметров.

Visual Basic
1

3
4

Dim feat1 As Feature
Dim face As Face2
Set feat1 = sel.GetSelectedObject6(1, -1)
Set face = sel.GetSelectedObject6(2, -1)
fm.FeatureExtrusion2 True, False, True, 0, 2222, 0.03, 222, False, 3, False, False, 131231, 23131, False, False, False, False, False, False, False, 1, 553532554, False

Третьим параметром(Dir) пишем true, чтобы вытягивание производилось в направлении противоположном нормали эскиза.

Шестой параметр (T1) – высота вытянутого элемента. Пишем 0,03 (30мм).

Третьим параметром с конца (T0) пишем 1 – вытягивание от указанной поверхности, грани или плоскости.
