Аргументы:

Option – опция – значение из перечисления swSketchTrimChoice_e;

X, Y, Z – координаты точки элемента эскиза.

Возвращаемое значение:

True – если элемент эскиза был обрезан.

swSketchTrimClosest – обрезать до ближайшего элемента эскиза;

swSketchTrimCorner – обрезать угол;

swSketchTrimTwoEntities;

swSketchTrimEntityPoint;

swSketchTrimEntities;

swSketchTrimOutside – обрезать снаружи;

swSketchTrimInside – обрезать внутри.
