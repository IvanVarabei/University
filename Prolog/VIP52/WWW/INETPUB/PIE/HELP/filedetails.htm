filedetails/9

filedetails(FName,Attr,Hour,Min,Sec,Year,Month,Day,Size)

Return system information about a file

Remarks

The arguments are as follows:

Filename A file name, indicating the names to match.

Attr The attributes to accept (see diropen()).

Hour,Min,Sec Time stamp.

Year,Month,Day Date stamp.

Size The size of the file.
