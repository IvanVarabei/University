New

Keyboard: F7, Alt+F+N

See also: Open, Save

The command New is used to create one of the following items: a dialog, a knowledge base or a text file. The type of the new item to be created is selected by clicking with the mouse on the corresponding button in the resulting dialog box:

Dialog Creates a new dialog

Knowledge base Creates a new knowledge base

Text file Creates a new text file

By pressing Enter or by clicking on the OK button, ESTA will create the new item. You will specify a file name for the new item when you close or save the item.
