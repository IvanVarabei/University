Visual Prolog DLL's for various examples

This directory contains Visual Prolog projects for constructing three DLL's that are used in various examples:

  • NONGUI
  • USEVPI (2 projects)

These projects are simply loaded into the Visual Prolog development environment and build (Alt-F9).

Result of building will be in EXE directory.

See also Using Visual Prolog DLL’s from other languages.
