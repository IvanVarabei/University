Search

Keyboard: Alt+E+S,F3

See Also: Search again

Use the Find Text dialog box to specify the text you wish to locate.

Text to Find

You enter the search string into the Text to Find input box. Choose OK to begin the search, or choose Cancel to terminate it.

Pressing the Alt+Down-arrow keys or pressing the List Buttons down-arrow will display the history list associated with theText to Find input box.

The search string can be specified by selecting a block of text before calling the Search Text dialog box. The selected text will appear in the Text to Find input box.

Options

This group of options defines the comparison method that the Editor uses when makes a search.

Direction

The Direction radio buttonsspecify the search direction.

Origin

TheOrigin radio buttons are used to specify whether the search should start from the current cursor position or from the beginning of the file.
