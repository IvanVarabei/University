Save

Keyboard: F2, Alt+F+S

See also: Save As, Open, New

This command is used to save one of the following items: the current dialog, the current knowledge base or the current text file.

Which item will be saved is determined as follows:

  1. If a text file window is the active window, the text file will be saved.
  2. If a dialog window exists, you'll be prompted to choose between the dialog and the knowledge base. The selected item will then be saved.
  3. The knowledge base will be saved.

The item will be saved with the existing name. If the item doesn't have a name, the Save As dialog box will appear in order to give the item a name. To give the item a name other than the existing one, you can choose the Save As command.
