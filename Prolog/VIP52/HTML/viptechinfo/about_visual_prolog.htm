About Visual Prolog

 

Visual Development of GUI applications

With Visual Prolog it has become easier than ever before to write state-of-the-art, graphical applications. You design and draw your user interfaces instead of programming them. The "Code Experts" will do all the trivial, and much of the complicated work for you! You can create a new functional application in a few minutes, and step by step enhance this from a prototype to your final application.

Safe and Fast

Visual Prolog meets the professional's demands for speed, performance and safety. It produces compact, fast and highly optimized native machine code, comparable to that of a C compiler.

Portable

In addition to this, your application is not tied to one operating system or one graphical platform. With Visual Prolog (Professional version) you can write portable applications across the MS-Windows 3.x, 95/98, NT and OS/2 platforms. And commandline applications (like CGI Scripts) are portable across DOS, Win96/98, NT, OS/2, SCO UNIX, Linux !

A Wealth of Facilities

All the components that make up the integrated environment can be used in your own programs. A wide range of standard predicates provides you with all the features you expect from a professional programming language. These include:

input/output procedures, 
arithmetic and string manipulation functions, 
external database system with B+trees and network support, 
operating system functions, and 
VPI (visual programming interface) layer which provides about 300 high-level GUI predicates, which make it possible to create portable GUI applications.
