Handling software licenses with an Intranet solution

Keeping track of which software license people are using in a company is not a trivial Task !

The following program implements a little database solution so people from the WEB browser can specify which software they are using on the Computers.

Feel free to try the App!

(Please do not spoil the database, you have full access to it !)

There are Four main Views:

List of All users.
List of All software
A View for a User.
A View for a Software

It is possible to jump around and modify the information for and update the information for these ! Also, for both Computers and software it is possible to specify what information to record for these, and what information to display in the table views.
