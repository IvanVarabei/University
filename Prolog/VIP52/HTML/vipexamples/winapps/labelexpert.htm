About the Label Expert

The Label Expert can be used to layout and print labels containing both text and graphics. The Label Expert is a very nice utility, written completely in Visual Prolog.

You have the possibility of choosing from a large number of predefined sizes of labelsheets, that are sold by manufacturers, or defining your own from scratch. From a programmers point of view, the application is interesting because it shows how to use custom controls, in a way that could be useful in many Visual Prolog projects.
