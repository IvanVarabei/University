The Web Solver

The Web Equation Solver (WES) is the Web Application tool, which gives the possibility to build numeric calculation models based on the Numeric Equations. The WES is task independent and depends just on the possibility to represent the task as the set of numeric equations. The programming of the calculation models and User Interface can be done just as the inclusions to the HTML file.

The magic facility of the WES is that it gives the possibility to make multidirectional calculations depending on the set of known (preset) and unknown (void) data.

The Implementation of the Web Solver based on the Equation Solver Tool, which is the universal tool and can be used in wide range of the user 's applications as the included Prolog code, as the DLL.
