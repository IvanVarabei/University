iMMap Pro Imagemap Demonstration
 
To try out the demo, wave the mouse over it and discover the embedded pop-up menus. Read about the features demonstrated here in the list below the applet.

You will need to activate java to view this menu.


About this demonstration
This shows some of the advanced features of iMMap Pro used to maximum effect.
This is just a small menu with 2 hotspots and 2 submenus with three items each. Each item could of course be hyperlinked to any number of HTML pages or other webpage functions.
Both the hotspots and menu entries all have mouse-over graphics defined (instead of the usual internal applet highlight markers).
In addition a submenu background has been used, and the usual text in the submenus has been replaced by tag graphics to allow the use of a more dramatic font.
The six submenu entries all dynamically load a contextually relevant image into the centre of the applet when highlighted by the mouse.
The images are automatically preloaded by the applet to ensure faster, smoother operation.
