The right tool for internet development!

Right now we are just about to drown in information on the web. What we need, are tools to help us navigate in this endless stream of information. Expert systems were very hype about 15 years ago, but generally failed to fulfill the high expectations. But now experts systems on the web can ignite the development of the true online society, where ordinary people work, trade, communicate and use the net as primary communication media.

Expert systems and rulebased applications can make order out of chaos and find and present only the relevant information to users. Visual Prolog is an obvious choice for developers of applications for the web, for instance "netbased"-expert systems, data mining of the net, intelligent Internet agents and more…, due to its rule based nature and good Internet support. Intelligent web sites -expert systems attached to homepages that can automate many processes, for instance support and help customers choose products, will play an increasing role in tomorrows information society. People get ready, developers start NOW!

Claus Witfelt, product manager of Visual Prolog: "The web needs qualitative improvements. In the 1st web generation all homepages were just posters. In the 2nd, the posters got linked together. The 3rd generation allowed us to attach applications to web-pages, but this still didn’t make the web as clear, searchable or usefull, on the contrary - just more complex.

In the next generation, web users demand tools to make things clearer, to make good searches and to use services, not just get loads of loads of unsorted poor-qualityinformations. We need programming tools to facilitate the new web users' demands, about guidance, automation, push-technologies and intelligent agents.

With Visual Prolog 5.0 we supply the necessary tools to move to the next 4th. web generation! We have already received lots of acknowledgements from users, because of all these new, exiting enhancements"

Visual Prolog has been used for medium- and large scale development for quite some years and has all this time been optimized for mission critical, high-security problems, typical planning and other AI-based problems but also more general propose applications.

Also the development environment features have gone through quite an enhancement! Visual Prolog developers aren’t limited in their design of user interfaces, neither their choice of platform. Visual Prolog is still one of the most portable development environments, now incl. web support. The visual development environment with new debugger supports the design process all the way. There are so many new features and instructive examples. Everybody should take a look at the free, timelocked version (www.visual-prolog.com)"
