Visual Prolog 5.0

Prolog Development Center announces version 5.0 of their Prolog Development environment, Visual Prolog. Visual Prolog 5.0 Professional includes a complete visual cross-development system for all 16 & 32 bit windows platforms plus 32-bit OS/2. Creates EXEs og DLLs, including ISAPI extension DLLs. Portable APIs for UI, SQL database connectivity, sockets, ftp, http, TCP/IP & CGI. Tools to parse and generate HTML, IPF & RTF. Includes debugger for 32-bit applications. Core language now includes Objects and Classes. Built-in linker can create executables for any supported platform from VDE running on anyother supported platform. Compiler and linker can be imbedded in your application with special runtime license (no runtime restrictions with ordinary programs). Hundreds of example programs, including a 'standard' Prolog interpreter, and a complete expert system shell (ESTA). Also includes textmode only versions of the compiler for LINUX and SCO UNIX.
