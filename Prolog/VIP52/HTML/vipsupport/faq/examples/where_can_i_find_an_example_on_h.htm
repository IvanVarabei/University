Where can I find an example on how to provide Internet support for an application developed in Visual Prolog?

You can find a nice example on how to provide Internet support for an application developed in Visual Prolog at http://www.edmgroup.com/edm_2000.htm.

Other examples can be found at the  page "Visual Prolog Demo Examples"

The <Visual Prolog root>\WWW sub-directory includes examples of Visual Prolog WEB applications. These examples can be tested on a local server. All examples are supplied with sorce code.
