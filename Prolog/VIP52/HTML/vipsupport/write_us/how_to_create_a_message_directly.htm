How to Create a Message Directly from Visual Prolog 5.1 VDE?

Visual Prolog 5.1 suggests registered users a very convenient way to contact with  PDC Visual Prolog Technical Support service while you work in the  VDE.
This is the most recommended way for sending questions about problems of using Visual Prolog.

  1. Open  the project concerning the problem.

  2. Choose the Help | Feedback to VIPSupport command.

  3. The "Create Mail  to PDC Support" feature will automatically:

create a letter,
insert the Visual Prolog technical support address
gather information that can be required for correct understanding of your problem e.g. Visual Prolog version, platform, project settings, etc.
  1. Give a short informative description of the problem in the Subject field.
    Please, do not use general subjects, such as: "Visual Prolog problem", "Help", etc.
  2. Type your product serial number in the Message field.
  3. Give a detailed description of the problem in the Message field.
    The information placed here should be sufficient to reproduce the problem on our computers; otherwise, the answer, probably, will be just:
    - "We cannot reproduce the problem!"
    Examples of code illustrating problems are required.
