Top level split window

There is little difference in the API of the top level and the bottom level. The main difference is that the top level sets defaults for the properties and sets the update and the resize callback.

So the top level API is the same as the bottom level API with a few additions. Most of the toplevel predicates can be implemented as simple calls to the bottom level predicates.

The prefix for the top level is "split"
