Visual Prolog Packages

New Split Window Tool

New Split Window Tool with an example has been added.
(<Visual Prolog root directory>\VPI\Include\SplitWin).

PDC's Document Handling Package

HTML 4.0 support has been added.
The Document Handling Package (DOC_TOOL) section of the online Help has been updated.
The structure of of the Document Handling package (DOC_TOOL) has been changed in Visual Prolog 5.2. Therefore, in the projects created by previous versions of Visual Prolog that use DOC_TOOL, it is necessary to  make manual changes. See Upgrade Notes
A bug  has been fixed.

Examples

New Project Files of Examples

Beginning with Visual Prolog v.5.2, we supply platform-independent <ProjectName>.PRJ project files for all examples instead of .VPR files. These files can be loaded by both  32-bit and 16-bit VDE versions. 

ESTA

The WEB version of ESTA now can be viewed under the Netscape Navigator 4.x.

New Examples

SplitWin example demonstrating Split Window Tool  has been added.  (<Visual Prolog root directory>\VPI\Toolexam\Splitwin.
New Parser examples have been added. (<Visual Prolog root directory>\VPI\Programs\Parser). 
<Visual Prolog root directory>\VPI\Programs\Parser\Doc sub-directory includes a new version of Parser documentation.
