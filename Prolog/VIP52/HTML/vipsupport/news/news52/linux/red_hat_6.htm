Red Hat 6.0 Support

Because the source code for the Linux kernel is freely distributed, a number of companies have developed their own distributions of Linux. In compliance with wishes of a number of our users, we have chosen the Red Hat 6.0 version of Linux to be supported in Visual Prolog 5.2. We have accomplished our tests under the Red Hat 6.0 version. We cannot guarantee that our VIP 5.2 Linux version will properly work under all Linux distributions.
