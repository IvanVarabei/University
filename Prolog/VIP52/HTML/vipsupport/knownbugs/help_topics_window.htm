Visual Prolog Technical Support

The "Help Topics" window is not initially activated when Win32 on-line Help is run.

In the current version of Visual Prolog the "Help Topics" window can be activated only by clicking on the "Help Topics" button in the Help window.

Workaround

To avoid the problem, it is possible to do the following modification: 

GLOBAL PREDICATES    
procedure integer WinHelp(window hWndMain, string lpszHelp, unsigned uCommand,
DWORD dwData) - (i,i,i,i) language stdcall as "WinHelpA"
HELP_FINDER = 11,
_R = winHelp(Win, "..\\HELP32.hlp", HELP_FINDER, 0),
....

This solution will work under  Win32 no matter whether winHelp32.exe exists on user's computer or not.
