How to Download the 1st Step Version Patch via FTP

You can download the patch directly from our FTP server.

To download Visual Prolog directly from the FTP server, use the following steps:

  1. Connect to ftp://ftp.visual-prolog.com as an anonymous user.
  2. Open  /pub/patches/p50to51 directory.
  3. Download s_50_to51.exe to upgrade Visual Prolog 5.0 1st Step version.
  4. Follow the "Installation instructions" beginning with the item 4.
