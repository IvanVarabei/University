Visual Prolog Technical Support

Patches and Downloads

Visual Prolog 5.2

The PDC does not provide a patch to upgrade Visual Prolog 5.1 Professional Edition to Visual Prolog 5.2 Professional Edition.

However, to evaluate Visual Prolog 5.2, you can download the Personal Edition of Visual Prolog 5.2. 

Visual Prolog 5.1 

You can download and apply an abbreviated  patch to upgrade core Visual Prolog directories  from the version 5.0 to Visual Prolog 5.1, build 565.
Note, please, that Visual Prolog 5.1 Professional is also available as a complete replacement instead of a patch. You can order a new Visual Prolog 5.1  CD  now.
 
You can also download Linux and SCO UNIX versions of Visual Prolog 5.1, build 565.
