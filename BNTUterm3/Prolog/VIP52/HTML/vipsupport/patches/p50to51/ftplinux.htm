How to Download the Linux Version of Visual Prolog via FTP

To download the Linux Version of Visual Prolog directly from the FTP server, use the following steps:

  1. Connect to ftp://ftp.visual-prolog.com as an anonymous user.
  2. Open  /pub/patches/p50to51 directory.
  3. Download linux.zip.
  4. Follow the "Installation instructions" in the README.TXT.
