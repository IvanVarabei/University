Q&A about Upgrading Visual Prolog and Downloading Patches

I use Visual Prolog 5.0. How can I receive the latest version of Visual Prolog?

We provide several possibilities to upgrade you Visual Prolog version:
 
If you are a registered user of Visual Prolog 5.0, you can download a patch to update the version 5.0 to the 5.1 version from this site. The update is free for all registered users - just install the new version on the top of the old registered version.
You can also  order a new Visual Prolog CD with the latest version of Visual Prolog.

I am upgrading my VIP version, and a patch file constantly produces errors. The error is:

error ept0009: Invalid Patch File 'E:\PROGUT~1\VIP\p_help.rtp'

Probably, this patch file has been corrupted during downloading. Compare the size of your file with that shown in the table.
If you obtain a different size, please, clean you cache and try to download the file once more.

I have checked the size of the downloaded file and found that it was wrong.  I  repeated downloading several times but always received the same wrong size. I think you provide an invalid patch file.

Please, check your cache, or contact your system administrator for help. Probably, you receive copies of the corrupted file from your cache, instead of downloading the correct file from our server.

While running a patch program the following message  appeared  in the "Results of Application of Patch File" list:  "Old Files Invalid.."

The message can be caused by two reasons:
  1. Some source files of the previous version (e.g. examples) have been changed by a user.
    A patch would not be applied to these changed files.
    Workaround:
    If you want to upgrade these files, please, restore them from the Visual Prolog CD, apply a patch, and then insert your changes if necessary.
  2. While Visual Prolog installation you have chosen an installation different from maximal. Hence, a patch program cannot find uninstalled files. You may pay no attention to this message if you have not changed the source files of the previous version (see item 1).

I use Visual Prolog 4.x. Can I upgrade it to the latest version using a patch?

No. You need  to upgrade your Visual Prolog ordering a new distribution.

Would be most welcome if Visual Prolog 5.1 would also be available as a complete replacement instead of a patch.
I find it rather difficult to apply these updates unless one has installed the whole original package.

Yes,  you can order a  Visual Prolog 5.1 CD. However, Visual Prolog 5.1 is also available as a patch on the WEB. Note, please, that the patch updates only a part of VIP directories.
