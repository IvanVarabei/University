How can I find the day of the week for a given date.

You can use a couple of predicates from our DATE package from ..VPI\INCLUDE\DATE directory.
Date package is a collection of  useful predicates for the conversion of different date and time formats into each other. It supplies an  application programmer with   methods to make different calculations with dates including support for the date and time format strings.
dt_date_to_offset(MY_Year, MY_Month, MY_Day, DATE_OFFSET)

Flow pattern: (i,i,i,o)

dt_offset_to_date(DATE_OFFSET, _Year, _Month, _Week, _DayOfMonth,
MY_DayOfWeek)

Flow pattern: (i,o,o,o,o, o)
