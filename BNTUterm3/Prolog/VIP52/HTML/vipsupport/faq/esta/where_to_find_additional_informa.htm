Where to Find Additional Information about the WEB Version of ESTA?

You will find a lot of useful information about ESTA WEB version in the "Introduction to Visual Prolog".  To read this Introduction, use the Help|Introduction to Visual Prolog menu command and then choose "Demos".

In the  "Visual Prolog Demo Examples " page, you will find a link to ESTA WEB pages under the heading "An Expert System on the WEB".

You will also find some useful tips in the  "Visual Prolog Knowledge Base".
