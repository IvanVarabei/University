Navigation  without Java Scripts

The picture database belonging to my project is incompatible with the new version of ESTA (5.1).

ESTA Picture Database is a Prolog external database itself, so the databases created in 16- and 32-bit versions of ESTA are incompatible.

If your picture database was created by 16-bit ESTA, you need to convert it into 32-bit format.

To fulfil this conversion, do the following:

Run 16-bit ESTA.
Convert the picture database into a text file with the help of Pictures | Save to text menu item.
Run 32-bit ESTA.
Restore the database from the text file, created by 16-bit ESTA, using Pictures | Consult from text menu item.
Note

ESTA stores pictures that are larger than 64 kilobytes in separate files. After converting a picture database, you can simple copy these files to a new location.
