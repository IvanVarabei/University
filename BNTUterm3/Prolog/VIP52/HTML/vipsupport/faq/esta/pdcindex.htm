Q&A about Expert System Shell (ESTA)

Can I create an Expert system using Visual Prolog?

Yes, Visual Prolog is supplied with the PDC expert system shell ESTA. The ESTA directory contains the executable file, full source code, and some implementation examples.

Can I interface ESTA to 3rd-party applications?

You can easily interface ESTA to 3rd-party applications like spreadsheets, databases, word processors etc. via the built-in DDE-interface.

Does ESTA include interface to Visual Prolog?

ESTA includes an interface to Visual Prolog, which means that you can go beyond the built-in functionality - you can build your own extensions to ESTA or integrate ESTA into an existing Visual Prolog application.

How can I get an overview of my knowledge in ESTA?

A knowledge base in ESTA has a hierarchical structure from which a tree display is automatically drawn. You can edit or expand your knowledge base directly from the tree. In this way you can easily get an overview of your knowledge.

Please, provide information  on how to include a picture in the ADVICE portion of a SECTION in ESTA?.

Unfortunately it is impossible in the current ESTA version.
