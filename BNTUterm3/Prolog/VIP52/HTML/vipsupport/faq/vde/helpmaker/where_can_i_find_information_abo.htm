Where can I find information about creating online Help system in Visual Prolog?

Visual Prolog Visual Development Environment (VDE) includes built-in Help Maker.

We recommend you the following information sources:

  1. Visual Prolog online Help (See: Visual Prolog Contents -> Visual Development Environment (VDE) ->Help Development System)
  2. "Creating Help  File" chapter of the "Getting Started" tutorial. This tutorial is supplied in two forms:
as a printed book
as a .DOC file.
You can find this file either on a VIP5.x CD
(RUN/DOC/ Getstart.doc), or, if you have checked the "Documentation" item, while installation, in the ../DOC directory on your HDD).
  1. You can also find some useful tips in the VDE section of the "Frequently Asked Support Questions" list.
    It is also possible to view Support FAQ directly from the VDE:
Use Help | Introduction to Visual Prolog menu command to open the "Introduction to Visual Prolog" at you disk and then choose Support -> Frequently Asked Support Questions.
Use Help | Visual Prolog on WEB menu command to see the advanced list at our WEB.
