Visual Prolog Technical Support

How to Find Answers about Visual Prolog Online?

Visual Prolog Technical Support provides the following online services to answer your technical support questions:

Use Help information provided with Visual Prolog
Inside your Visual Prolog you'll find a powerful online Help system that answers many questions about using Visual Prolog.
To use Help, start Visual Prolog, and then click the Help menu on the toolbar.
Another way to use Help  from Visual Prolog VDE is to press:
CTRL+F1 to open Help Contents
F1 to receive Local Help. 

You can also read "Introduction to Visual Prolog" (Intro.htm) in the Visual Prolog root directory.
Visual Prolog 5.1 and 5.2 provides reading of the "Introduction to Visual Prolog" by choosing Help | Introduction to Visual Prolog command.

Read Visual Prolog documentation
Visit Visual Prolog WEB pages
Visual Prolog 5.1 and 5.2 suggest a direct access to the Visual Prolog WEB sites while you work in the  VDE.
The Help | Visual Prolog on the Web command  provides additional commands that take you to helpful Visual Prolog Web pages.
Browse Frequently Asked Questions and Answers (Q&A)
See answers to frequently asked questions about Visual Prolog.
Browse a "Knowledge Base"
Read Technical Notes
Under Technical Notes we place information about some common tasks we have been asked about, which were not properly documented in the current Visual Prolog version.
Interact with other VIP Users
You can join Visual Prolog Discussion Forum
Try to locate your problem yourself using "The Universal Troubleshooting Process" described at the Troubleshooters.Com(R).
Download a Patch
If you are having a problem with the Visual Prolog system, you might find that it is solved in the last version of Visual Prolog. You can download and apply a patch to upgrade your Visual Prolog. 
Send your question to the Visual Prolog Support TEAM.
If a description of you problem cannot be found in the Knowledge Base, and you have exhausted other possibilities, it is possible to ask the Visual Prolog Support team for help. Note, please, that the e-mail support is available only for the users of the Professional Edition of Visual Prolog.
