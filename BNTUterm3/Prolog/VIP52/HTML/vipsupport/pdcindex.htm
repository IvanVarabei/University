Visual Prolog Support

Welcome to Visual Prolog Support. The purpose of this web is to enhance the support services we provide to our customers. We have provided a number of resources here to help you resolve your problems.

Frequently Asked Support Questions

We hope that a list of the most common support question and answers will help you to solve your problems.

Updates to Visual Prolog

If you are having a problem with the Visual Prolog system, you might find that it is solved in the last version of Visual Prolog. It is possible to download and apply a patch to upgrade your Visual Prolog or order a new CD copy of the latest version of Visual Prolog.
