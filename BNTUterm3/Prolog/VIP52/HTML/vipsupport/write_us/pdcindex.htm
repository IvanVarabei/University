Contact Visual Prolog Technical Support Team

support@visual-prolog.com

If you're experiencing a problem with  Visual Prolog, and an answer cannot be found in the online Help system provided with Visual Prolog, or  on this WEB,  it is possible to ask the Visual Prolog Support team   for help.

Visual Prolog Support team is responsible for answering questions concerning problems of using Visual Prolog language, VPI, VDE, and other tools, and packages supplied with Visual Prolog.
Note please that the e-mail support is available only for the users of the Professional Edition of Visual Prolog.

You are welcome to contact us in two different ways:

Create a message directly from Visual Prolog 5.1 VDE

Send a letter to the Visual Prolog Support team

Please, before sending a letter read: How to Find Answers Online?  Maybe, it will help you to find answers to your questions.

If you have questions concerning purchasing or upgrading of Visual Prolog, contact the sales department.

If you have developed an application in Visual Prolog, and have a description of it on the WEB, we will be happy to include a link to this description! E-mail: leo@pdc.dk
