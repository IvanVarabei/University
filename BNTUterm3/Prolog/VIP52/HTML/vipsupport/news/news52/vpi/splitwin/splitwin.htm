Split Window Tool

The splitwin tool has mainly been designed by Carsten Kehler Holst from PDC.

A split window is a window split into a grid of panes each containing a child window. The split window has resize bars between the child windows. The relative size of the child windows can be adjusted by dragging in the resize bars.
