About VIPsupport

VIPsupport is responsible for answering questions about problems with the use of the Visual Prolog language, VPI, VDE, and other tools and packages supplied with Professional Editions of Visual Prolog.

Guidelines for  Using  the Visual Prolog Support Service

Personal Edition Users:

Note that your "PDC END USER SOFTWARE LICENSE AGREEMENT" requires;
that you accept: "You cannot get free support for the Personal Edition."

Professional Edition Users:

Before sending a support request, please read the guidelines below:

Your message should contain:

  1. Your "Product serial number".
  2. The information generated by using the "Help | Create Mail to PDC Support" menu option (available in 5.1 and 5.2 versions of the VDE).
    Support requests containing the above information receive a higher priority.
    If your VDE doesn't have the "Create Mail to PDC Support" menu item, please supply the following information:
    OS, VDE version, compiler and linker options, project settings etc.
  3. Use a short, precise, and relevant title (Subject line). For example: 
    "Classes: cannot use nocopy single facts with binary domain".
  4. One Question per message please.
  5. Please check the following information sources before submitting your e-mail:
    How to Find Answers about Visual Prolog Online?
    Questions and Answers (Q&A)
    Visual Prolog Knowledge Base
    Visual Prolog Discussion Forum  

  6. Much of the above information can be found in  HTML\vipsupport\KNOWNBUGS\pdcindex.htm,  HTML\vipsupport\FAQ\pdcindex.htm, and HTML\vipsupport\TechNotes\pdcindex.htm in your Visual Prolog installation.

Please do not ask VIPSupport about general problems like:  - Visual Prolog and Fuzzy Logic?
For such questions we recommend Visual Prolog Discussion Forum.

Please follow the above guidelines when you E-mail a support request.
