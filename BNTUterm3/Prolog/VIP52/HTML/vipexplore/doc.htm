Documentation

The Visual Prolog documentation consists of about 900 pages of printed documentation separated into four books: Language Tutorial, Visual Development Environment, Getting Started and Visual Programming Interface. In addition, a comprehensive on-line help system is included with the system. You will find the documentation  for Visual Prolog in the DOC subdirectory (doc. for release 5.0 is the latest version). The documentation is supplied as MS Word 97 files. If you do not have a copy of Word 97, you can install the Word viewer from the CD or download a copy from the Microsoft WEBSite.

DOC\GETSTART.DOC - Getting Started with Visual Prolog - overview.
DOC\LANGUAGE.DOC - The Prolog Language Tutorial
DOC\VDE.DOC - The Visual Development Environment.
DOC\VPI.DOC - The Visual Programming Interface.

Note: The Visual Prolog documentation can be ordered separately from the Prolog system !
