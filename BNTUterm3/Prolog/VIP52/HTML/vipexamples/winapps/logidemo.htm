A demo of a tool for logistics, transportation and distribution

Download.

Logidemo is a demo of an application for planning and optimizing   transportation, production, distribution and storage utilisation. Logidemo enables the planner to simulate changes in key parameters of the total production plan e.g. distribution costs, start up costs, etc.

Run the demo, and start by filling out values for some or all of the windows in the Forms-menu. Then you can generate a plan and examine the plan graphically in the Map/View Map-menupoint or generate a report (in the report-menu).

Logidemo is quite similar to some of the programs, actually developed for real-world purposes by the consultancy-department in Prolog Development Center. Read more about LogiPlan or about other of PDC's planning applications.

The planningmotor in Logidemo is developed in Visual Prolog. It can deliver a solution even to quite complicated problems in a few minutes.
