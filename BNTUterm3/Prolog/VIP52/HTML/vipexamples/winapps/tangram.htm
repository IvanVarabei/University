Tangram - an Ancient Oriental Game

Download.

Tangram is an ancient oriental toy which has 7 pieces, 5 triangles in
different sizes, 1 square and 1 diamond. The objective of the game is
to form a given shape using all 7 pieces.

Thousands of shapes can be made by moving the different pieces
around. This puzzle game is especially great for children in order to
practice their matching skills. You may think this is only a game for
children, but you will find it challenging once you try it.
