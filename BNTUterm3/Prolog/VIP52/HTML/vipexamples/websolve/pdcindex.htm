The Web Solver

The Web Equation Solver (WES) is the Web Application tool that makes it possible to build numerical calculation models based on numerical equations. The WES is task independent and depends only on the possibility of representing the task as a set of numerical equations. Programming of the calculation models and the User Interface can be done just as inclusions to a HTML file.

The magic facility of the WES is the possibility of making multidirectional calculations depend upon a set of known (preset) and unknown (void) data.

Implementation of the Web Solver is based on the Equation Solver Tool,  the universal tool that can be used in a wide range of user applications as the included Prolog code, as the DLL.
