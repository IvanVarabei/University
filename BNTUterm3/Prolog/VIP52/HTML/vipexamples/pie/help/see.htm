see/1

see(Filename)

Open file for input.

Remarks

see/1 opens the disk file FileName for reading and attaches the input stream to it.

Example 

tell("dd.txt"),
write("line 1\n"),
write("line 2\n"),
write("line 3\n"),
told,
see("dd.txt"),
readln(Str1),
seen,
display(Str1).
