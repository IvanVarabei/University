Integrated, state-of-the-art Help Maker

The built-in help authoring system makes it very easy to give your application on-line help. The Help system is based on the PDC Hypertext Abstract Machine (HAM). In the Help authoring system it is possible to enter the help text interactively, mark new links with the mouse, and to follow existing links during the design phase. The help system can output both the Windows .RTF format and the OS/2 IPF format so it is possible to generate native help systems for both OS/2 and Windows.

The Help compiler HC.EXE is not included with Visual Prolog, but can be found in the Visual C++ and the Borland C++ products. If you need a newer version of the Win3.1 help compiler it may be worth checking out the Softlib directory at ftp.microsoft.com or trying the following URL: ftp://ftp.microsoft.com/Softlib/mslfiles/hc505.exe.
