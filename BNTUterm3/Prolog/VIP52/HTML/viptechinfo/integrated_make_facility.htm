Integrated Make Facility

The Visual Prolog Make facility takes care of the compilation, resource file generation, code updating and linking. The compiler will automatically generate include file information for the Make facility, so it knows only to regenerate/ update/ compile/link the necessary files.

The make facility is very general, so you can make any changes to the scripts, and you can actually use the Visual Prolog environment for compiling and linking C programs - or your own new language.

The make facility is tightly integrated with the tree display.
