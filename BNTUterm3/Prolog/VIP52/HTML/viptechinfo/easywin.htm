EasyWin - Windowing Applications without Event Handling

Selecting "EasyWin" from the application expert's UI Strategy menu provides an easy way to build quick windows applications. These applications can use the various textmode read and write predicates as if for standard input/output, and a scrolling text-window is automatically created within the host GUI to support this. This removes all the complexity of working with event-handlers, but allows your application to take advantage of the large memory space available under Windows.
