CGI Support & Examples

Visual Prolog is a powerfull tool when it comes to generating dynamic WEB pages. Generating a CGI Scripts in Visual Prolog is very easy, the HTML can be generated by just using write statements.

The CGI Scripts can be used for many different purposes, from Database interfaces, expert systems, e-comerse, workgroup management or a WEB interface to your special application!

Take a look at our WEb to see a number of examples.
