NEW RELEASE: VISUAL PROLOG 5.1

Visual Prolog 5.1 fixes some problems and contains some nice new features.

The predicate mode checking is now fully implemented, and the compiler can now do a great job checking procedural programming where procedures are assumed not to fail.
New examples demonstrating how to interface Visual Prolog with Java, Visual basic, Visual C++ and Delphi have been added.
The debugger has been considerably improved; it is now more tightly integrated with the VDE.
Access to the Visual Prolog WEB sites and PDC Technical Support services from within the VDE.

For more details
