Installing Visual Prolog

Windows 95/98/NT/2000: Run SETUP32.EXE from the INSTALL directory on the CD. If you are using MS-Explorer you can just click on the link below (select "Run this program from its current location")




OS/2: Run SETUPOS2.EXE from the INSTALL directory on the CD
Linux or SCO: You should look in the SCO and LINUX subdirectories, and follow the instructions in the read me files there.
Windows 3.1: You must manually copy the RUN directory from the CD. Sorry we do not supply a Win16 installation program!

Visual Prolog does not install any .DLLs and modifies no existing .INI files.  You can remove Visual Prolog entirely from your computer by deleting the directory where it is installed!

It is possible to run Visual Prolog from the RUN directory on the CD.
