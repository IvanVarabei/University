Ctrl+X,Shift+Del Cut

Ctrl+U

Changes selected text to all uppercase letters

Ctrl+O

Changes selected text to all lowercase letters

Ctrl+P

Toggle case. Changes all uppercase letters to lowercase letters and all lowercase letters to uppercase letters in the selection.
