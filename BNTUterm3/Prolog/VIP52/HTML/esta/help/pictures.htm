Pictures Database

Keyboard: Alt+P+P

See also: Pictures in ESTA

This command is used to modify or view the pictures database. This invokes the Picture Database Dialog

{bmc PicDB_1.bmp}

This dialog is devided into two parts. The pictures stored in the Pictures Database are listed in the box at the left side of it. And the currently selected picture is shown at the right side. The Zoom button at the right bottom corner is used to display the picture on the full screen.

The dialog has three tabs:

  • Database
  • Attributes
  • Hotspots
