Selecting text with mouse

You can use the mouse to select text. In order to select any piece of text hold down the left mouse button, drag mouse across the text you want to select and release the button.

To select a word double-click the word.

Using SHIFT key with mouse:

  • If there is no selected block in the active Edit window. Then click the left mouse button in an appropriate position to select the beginning of a block, press Shift key and hold it down, move mouse cursor across the text you want to select and again click the left mouse button.
  • If there is a selected block in the active Edit window. When you press Shift key and click the left mouse button, then you select the new text block with the same beginning position as the previous one 1 and with the end position where the mouse was clicked.
