The WINBIND Directory

This directory contains the direct bindings to the MS Windows 16 bit API supplied with the previous version of PDC Prolog.

Directory Content
[DEMOS] Some demo programs
[DLLDEMO] An example of producing a Prolog .DLL
[INCLUDE] Header files for WINBIND
[LIB] The needed libraries
[SOURCE] Source code for the bindings
[TOOLS] A couple of tools from the manual

It is possible to use these bindings in conjunction with the VPI libraries. You can include the .CON, .DOM and the .PRE files in VPI Programs.
