The ODBCBIND Subdirectory

ODBCBIND is a layer between application program written in Visual Prolog and Microsoft Open Data Base Connectivity (ODBC). ODBCBIND includes full bindings to the ODBC API allowing Windows only applications taking of all the features in the ODBC API.

File Purpose
[INCLUDE] This directory contains the header files.
[LIB] A library to support the conversions: ODBCLIB.LIB, with various subdirectories for the different platforms !
[SOURCE] Source code and makefiles to Build the ODBCLIB library.
[EXAMPLE] Some examples of using ODBCBIND.
[DOC] Documentation for ODBCBIND.
