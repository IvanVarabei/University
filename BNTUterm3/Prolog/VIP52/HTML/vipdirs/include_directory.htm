The INCLUDE Directory

The files in the global include directory are some include files that are not specific to the VPI, WINBIND, SQLBIND etc. Some support the old text mode windowing system found in previous versions of PDC Prolog:

File Purpose
IODECL.CON Some constant declarations to do portable file handling
IODECL.PRE Can be used to redirect the default output from write.
ERROR.CON Constant definitions for error codes in VPRO
ACS.CON UNIX key definitions
EDITHOOK.PRE Hook into the old text editor
ERRHND.PRE Error routine for old text mode editor
READKEY.CON Constant definitions for keys
VARARG.DOM Domain definitions for a variable argument call tool
VARARG.PRE Predicate declarations for vararg call (See the file)
OSERRM.PRE Can be used to change the default error handling under Win.
