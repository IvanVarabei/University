Information about License control program

The program uses just a simple Prolog Consult file, but it can easily be connected to a SQL server.

  • Source Code for main program.
  • Source code to database declarations.
  • Source code to database access implementation.

Download source code for the Program

If you want to use the program in your company, you should:

  • Download the source code.
  • Compile it to an executable.
  • Place the executable in a directory under the MS Information Server which has Execute permissions.
  • Create a directory: "c:\\licenses"  the program stores its database here. (you can just modify it if you prefer an other location).
