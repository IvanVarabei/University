Open

Keyboard: F8, Alt+F+O

See also: New, Save , Save As

This command is used to open an existing dialog, knowledge base or text file. Selecting a type in the combobox: List Files of Type, will cause a listing of files with the selected extension from current directory in the file list box. The types of files supported by ESTA are:

Extension Type

*.kb Compiled knowledge base files

*.kbt Text knowledge base files

*.log Dialog files

*.txt Text files

In the File Name dialog box you type or select from the list box the filename you want to open. To list other files type: an asterisk *, a period, and a three-character extension. For instance, if you want to see files with a ".def" extension in a directory type: *.def. All files can be seen by entering *.* .

The drive and the directory used to search for and open the file can be changed in the Drives: and Directories: boxes.

If you open a file with an extension other than those mentioned above, ESTA will attempt to treat it as a text file.
