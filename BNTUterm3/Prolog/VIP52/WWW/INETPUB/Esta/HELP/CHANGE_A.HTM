Change All push button

Choose Change All if you want the editor to replace all occurrences of the search string defined by the Options, Direction,

and Origin options.

If the Editor finds the specified text and if you checked Prompt on replace option, the editor will ask you if you want to make the replacement.

You can answer 'Yes', 'No, 'All' or 'Cancel'. 'All' means that all remaining replacements will be done without further prompting. 'Cancel' interrupts the search.
