Cursor movement commands

In order to set the insertion point using the mouse, position the mouse cursor at the desired location and click the left mouse button.

Command Key

Character left Left

Character right Right

Word left Ctrl+Left

Word right Ctrl+Right

Line up Up

Line down Down

Page up PgUp

Page down PgDn

Beginning of line Home

End of line End

Top of file Ctrl+Home

Bottom of file Ctrl+End
