Picture Field

The picture field is used to specify a picture, that will be showed in connection with the parameter. The picture can be specified by a name from the picture database in qoutes or by a parametername without quotes. In the latter case the value of the specified parameter determines the picture, that will be showed. The parameter value must be identical to a name in the picture database in order to show the picture.

The syntax for the picture field is:

<picture field> ::= picture <parameter name> | '<picture name>'

Examples

picture car

picture 'sedan'
