Origin

The Origin options determines where the search begins

Options This group of options defines the comparison method that the Editor uses when making a search.

Direction The Direction radio buttonsspecify the search direction.

Origin The Origin radio buttons are used to specify whether the search should start from the current cursor position or from the beginning of the file.

From Cursor

The From Cursor option starts the search at the cursors current position.

The search then proceeds either forward to the end of the file, or backward to the beginning of the file (depending on the Direction setting).

Entire Scope

The Entire Scope option searches the entire file (no matter where the cursor is positioned in the file).
