Save As

Keyboard: Alt+F+A

See also: Save, Open, New

This command is used to save and specify the name of any of the following items: the current dialog, the current knowledge base or the current text file. To save an item under its existing name, simply choose the Save command.

When the Save As dialog box appears, type a new file name in the file name box or select one of the files in the list box. In the latter case, the existing file will be overwritten if you choose to do so. If no extension is specified, the selected extension from the List Files of Type box will be added automatically. The types of files supported by ESTA are:

Extension Type

*.kb Compiled knowledge base files

*.kbt Text knowledge base files

*.log Dialog files

*.txt Text files

The drive and the directory where the file is to be saved can be changed in the Drives: and Directories: boxes.
